# 🌸 TUHFA Arts – Full Stack Web Application

TUHFA Arts is a modern full-stack web platform built for showcasing artistic products and managing custom orders in an elegant and structured way.
The project is designed with scalability and real-world business logic in mind.

---

## 📌 About the Project

This platform allows users to:

* Browse artistic collections and products
* Submit custom design requests
* Track order status
* Create accounts and log in securely

It also provides a backend system to manage users, orders, and authentication in a clean and organized structure.

---

## 🧠 Tech Stack

The project is built using modern web technologies:

Frontend:

* React (Vite)
* JavaScript / TypeScript
* CSS

Backend:

* NestJS
* Node.js
* TypeScript
* JWT Authentication

Database:

* MongoDB

---

## ✨ Features

* User authentication system
